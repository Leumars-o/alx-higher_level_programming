Using 2-matrix_divided.py

This module defines a matrix division function ``def matrix_divided(matrix, div)``.

Usage:
=====

``matrix_divided(...)`` returns a new matrix (new_matrix) with each element of the matrix divided by 'div' and rounded  to 2 decimal places.

::

    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided
    >>> matrix = [
        [1, 2, 3],
        [4, 5, 6]
                ]
        print(matrix_divided(matrix, 3))
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
